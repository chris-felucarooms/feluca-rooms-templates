README – Feluca Rooms Tenant Templates
📘 Feluca Rooms – Tenant Communication Templates
This repository contains all official templates used for communication with Feluca Rooms tenants.
All documents are written in English, as they are intended for international students and guests.

The goal is to ensure clarity, consistency, and professionalism in every interaction, while reducing the time required to manage repetitive communication.

📂 Repository Structure
1. Check-in and Check-out Templates
Templates used before, during, and after the tenant’s stay:

check-in-instructions.md

check-out-instructions.md

welcome-message.md

2. House Rules and Guidelines
Documents that explain how to live in the apartment:

house-rules.md

waste-collection.md

wifi-instructions.md

3. Support and Emergency Templates
Templates for reporting issues and handling urgent situations:

maintenance-request-template.md

emergency-instructions.md

🎯 Purpose of This Repository
Provide clear and standardized communication to all tenants

Reduce misunderstandings and repetitive explanations

Support automation workflows (Make, WhatsApp templates, Feluca Rooms app)

Ensure a consistent experience across all apartments and semesters

Maintain a professional and international-friendly communication style

🔧 Integration with Automations
These templates are designed to be used directly by:

automated check‑in flows

automated check‑out reminders

maintenance ticket creation

WhatsApp message sequences

the future Feluca Rooms app

Each template is written in a modular way to allow easy reuse and automation.

📑 Updating Templates
When updating a template:

keep the tone simple, clear, and friendly

avoid unnecessary details

maintain English as the only language

update the version number inside the file (if present)

commit with a clear message describing the change

🧩 Relation to Other Repositories
feluca-rooms-docs
Contains all internal Standard Operating Procedures (SOPs) in Italian.
This repository (feluca-rooms-templates) is the tenant-facing counterpart.

feluca-rooms-automations (future)
Will contain automation flows that use these templates as message blocks.

📌 Notes
All templates must remain in English.

No operational details should be added here (those belong to the SOPs).

Templates should be short, clear, and easy to understand for non-native speakers.
